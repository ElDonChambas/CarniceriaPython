NOTAS:
Mantener siempre el ejecutable (.EXE) en una carpeta donde exista "bases_de_datos" con sus archivos .CSV (artículos.csv, familias.csv, pedidos.csv, unidades.csv).

He agregado por seguridad que, si no se encuentran, que el programa los cree pero, pedidos.csv y articulos.csv se crea vacío, unidades.csv y familias.csv se crean con unos datos por defecto; pero lo recomendable es NO BORRAR o CAMBIAR EL NOMBRE A LOS ARCHIVOS NI LA CARPETA.

Si tienen dudas, pueden contactarme siguiendo los enlaces que dejé al inicio de este archivo, adjunto el código fuente y los archivos necesarios.

Las librerías usadas son:

PyQt5: para la interfaz
Pandas: Para el manejo de las bases de datos en CSV
Reportlab: Para crear los PDF para imprimir con el diseño exacto al que fue solicitado.

Mando el ejecutable y el código fuente con los archivos .CSV que usé para hacerle pruebas, pueden usarlo para las suyas también ya que tiene pedidos con diferentes fechas de ingreso y entregas.

Algunos pedidos los hice cuando aún no había validaciones, así que puede haber datos un poco raros, pero que no se pueden dar ahora que todo está controlado. 

Si quiere empezar a agregar datos desde cero, puede simplemente borrar o renombrar la carpeta "bases_de_datos" que está en la misma ruta que el ejecutable o el archivo .py (dependiendo de cuál esté abriendo) para comenzar con archivos desde cero.
